# Content Editor Guide (Sanity CMS)

This guide explains how to update the `SEO Optimizer` marketing website.

## 1. Login to Studio
*   Go to `https://your-project.sanity.studio`
*   Login with your provider (Google/GitHub).

## 2. Editing Content
On the left sidebar, click **Landing Page**. There is only one global document.

### Hero Section
1.  Locate the **Hero Section** field group.
2.  Update **Headline** and **Hero Image**.
3.  Changes autosave as "Draft".
    *   ![Screenshot: Hero Editor Fields](placeholder)

### Features
1.  Scroll to **Features / Benefits**.
2.  Click "Add Item" to create a new benefit block.
3.  Drag items using the handle on the left to reorder them.

### Pricing
1.  Find **Pricing Plans**.
2.  Edit price, tier name, or feature lists.
3.  Toggle "Is Popular?" to highlight a plan.
    *   ![Screenshot: Pricing Table Editor](placeholder)

### FAQs
1.  Scroll to **Frequently Asked Questions**.
2.  Add/Edit question and answer pairs.

## 3. Publishing
*   When finished, click the **Green "Publish" Button** at the bottom right.
*   **Wait ~2 minutes**: Vercel will automatically rebuild the site.
*   Refresh your live website to see changes.

## 4. Previewing Drafts
*   (If configured) Click "Open Preview" in the "Production" dropdown at the top to see changes before publishing.
